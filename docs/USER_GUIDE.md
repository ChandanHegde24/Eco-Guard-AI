# User Guide

## Typical Workflow

1. Open the dashboard.
2. Select latitude, longitude, and two dates.
3. Click "Run AI Change Detection Analysis".
4. Review risk level, recommended action, and index deltas.
5. Check recent runs in the timeline panel.

## Interpreting Risk

- Green: low observed change.
- Yellow: moderate change, monitor closely.
- Red: high change, immediate action recommended.

## Troubleshooting

- If analysis is unavailable, verify Earth Engine authentication.
- If frontend cannot connect, verify backend is running on port 8000.
